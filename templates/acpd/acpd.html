Análisis de Componentes Principales (PCA)

El análisis de componentes principales es un algoritmo útil para reducir la cantidad de variables de conjuntos de datos, mientras se conserva la mayor cantidad de información posible.

A continuación se mencionan los pasos para realizar PCA:

  • Paso 1: Evidencia de variables posiblemente correlacionadas.
  • Paso 2: Estandarización de los datos.
  • Paso 3: Se calcula la matriz de covarianzas / correlaciones.
  • Paso 4: Se calculan los eigenvectores y eigenvalores.
  • Paso 5: Se decide el número de componentes principales.
  • Paso 6: Se examina la proporción de cargas.

De manera similar como ocurre con el PCA, el ACD es útil para reducir el número de variables, de un espacio de alta dimensión a uno de menor número de dimensiones.
